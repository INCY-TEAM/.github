### Supported protocols

`VLESS` `Reality` `VMess` `Trojan` `Shadowsocks` `Hysteria2` `SOCKS5` `WireGuard`

### Features

- Subscription-based server management with auto-updates
- Flexible routing profiles — split tunneling by domain and IP rules
- Full xray JSON config import (observatory, balancers, custom geo)
- TLS fragmentation for censorship circumvention
- Real-time connection statistics and logs
- No data collection — everything stays on your device
